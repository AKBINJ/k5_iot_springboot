3. 기능 분담 의견 나누기


4인

- **A(리드/백엔드)** - 김승민
: 보안/JWT, [ User 도메인 + 개인정보수정 / 프로필 ]  , [ Tag 도메인 - 다대다 설정 ]

- **B(백엔드)** - 오신혁
: DB 세팅 + Entity 파일 생성
  , [ Task 도메인 + 필터(상태/태그/담당자/기간) ], baseTimeEntity


- **C(백엔드)** - 박진영
: Controller 파일 생성 (+ 기본 어노테이션 첨부 및 ApiMappingPattern 등록 / 경로 메모 첨부)
  , [ Comment/Notification(공지) + 필터링/정렬 ],
: common 세팅(ApiMappingPattern/ENUM타입/DateUtils 등)
 



- **D(백엔드)** - 정지훈
: Service/ServiceImpl 파일 생성 (+ 기본 어노테이션 첨부 및 파일 상단 권한 메모 첨부)
 , Repository 파일 생성 및 handler(GlobalExceptionHandler 설계)
  , [ Project 도메인 + 필터링/정렬 ]
